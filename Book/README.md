# Book Haven - Online Book Store

A fully functional, responsive book store website built with HTML, CSS, and JavaScript.

## Features

### 1. **Modern Responsive Design**
- Clean, professional interface with a book-themed color scheme
- Fully responsive layout that works on mobile, tablet, and desktop
- Smooth animations and transitions

### 2. **Book Catalog**
- Display of featured books with cover images
- Book details including title, author, price, and category
- Search functionality to find books by title, author, or category

### 3. **Shopping Cart System**
- Add/remove books from cart
- Adjust quantities of items in cart
- Real-time cart total calculation
- Cart modal with item details

### 4. **User Interface Features**
- Interactive navigation with smooth scrolling
- Book details modal with full descriptions
- Notification system for user actions
- Newsletter subscription form
- Category browsing section

### 5. **Additional Features**
- About section with store statistics
- Contact information and social media links
- Responsive footer with newsletter signup

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with CSS Grid, Flexbox, and CSS Variables
- **JavaScript (ES6+)**: Interactive functionality and DOM manipulation
- **Font Awesome**: Icon library for UI elements
- **Google Fonts**: Merriweather and Open Sans fonts
- **Unsplash**: High-quality book images

## How to Use

1. Open `index.html` in any modern web browser
2. Browse the featured books in the main section
3. Use the search bar to find specific books
4. Click "Add to Cart" to add books to your shopping cart
5. Click the cart icon to view and manage your cart
6. Click "View Details" to see more information about a book
7. Use the navigation menu to explore different sections

## Key JavaScript Functions

- `renderBooks()`: Displays books in the grid layout
- `addToCart()`: Adds books to the shopping cart
- `updateCartUI()`: Updates cart count and total
- `showBookDetails()`: Displays book details modal
- `handleSearch()`: Filters books based on search input
- `showNotification()`: Shows temporary notifications
